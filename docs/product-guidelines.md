# Product Guidelines

## 1. Design System: "Swiss Red"
- **Overall Aesthetic:** Professional, clean tech website.
- **Color Palette:**
  - **Base:** Black, white, or greyscale (monochrome).
  - **Highlight:** "Swiss Red".
- **Typography & Layout:**
  - **Strict Grid System:** Layouts must be built on a strict grid system, with the ability to import grid configurations across every page to maintain consistency.
  - **Geometry:** All clickable elements and components must have **NO** border radius (strict right angles).
  - **Outlines:** Key content areas must be outlined with thin, solid black lines.
  - **Navigation:** A modern, blurred, fixed floating navigation bar that sits above the content.
- **Component States:**
  - Interactions should feature strong, high-contrast focus states (e.g., inverse colors or thick outlines).
  - Include subtle animated feedback (e.g., sliding or scaling slightly) upon user interaction.

## 2. Copywriting & Tone of Voice
Tone should adapt based on the context of the page:
- **Main/Landing Page:** Visionary and inspiring, highlighting the future of the company and its capabilities.
- **Production/Tools Page:** Professional, direct, and focused on technical specifications (e.g., the 3D molecule viewer and IDE).
- **Contact/Blog Pages:** Approachable and friendly to welcome visitors.

## 3. Localization
- **Languages Supported:** English and Simplified Chinese.
- **Strategy:** Auto-detect the user's browser language on initial load, providing a persistent and clear manual toggle for the user to switch languages at any time.

## 4. User Experience (UX) Priorities
Ranked in order of importance:
1. **Visual Impact:** High-quality presentation, crisp layout transitions, and strict adherence to the sharp "Swiss Red" aesthetic.
2. **Information Density:** Efficiently packing useful data and links on screen, utilizing the grid system effectively.
3. **Speed & Performance:** Ensuring load times are fast and assets are optimized, while maintaining the required visual fidelity.
